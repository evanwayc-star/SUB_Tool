# SUB Tool — 上字幕工具

Arctime 風格的上字幕工具，**同一份 `index.html` 同時支援網頁版與桌面版**。

| 版本 | 啟動方式 | 適合 | 媒體能力 |
|------|----------|------|----------|
| **桌面版（推薦）** | 雙擊 `啟動桌面版.bat` | 完整功能 | **MXF / 多音軌同時播放 / 數 GB 長片**（系統 ffmpeg） |
| 網頁版 | 雙擊 `啟動工具.bat` | 免安裝、快速 | MP4/MOV/MP3/WAV 原生；MXF 受瀏覽器記憶體限制 |

---

## 一、桌面版（解除所有限制）

### 安裝與啟動

1. 先安裝 [Node.js](https://nodejs.org)（一次性）。
2. 先安裝 [ffmpeg](https://ffmpeg.org)（一次性，MXF/多音軌需要；放入 PATH 或 `C:\Program Files\FFMPEG\bin\`）。
3. 雙擊 **`啟動桌面版.bat`**。第一次會自動 `npm install`（下載 Electron），之後即可直接開啟。

> 桌面版啟動時會自動偵測 ffmpeg；若未安裝，原生 MP4/MOV/MP3/WAV 與字幕編輯仍可使用。

### 桌面版多了什麼

- **直接讀 MXF / MKV / AVI 等非原生格式**：自動用系統 ffmpeg 轉一支 720p 代理預覽片（畫面），原音軌另外抽取保持原質。
- **多音軌「同時」混音播放**：每條音軌抽成暫存檔，各自以串流方式播放並與影片同步 → 可同時聽到所有音軌，或單獨獨奏（S）、靜音、調整音量。**不受記憶體限制，長片也可以。**
- **整片波形**：以低取樣（8kHz 單聲道）由 ffmpeg 產生整部片的波形，記憶體友善。
- **專案 `.subtool` 記住影片完整路徑**：重新開啟專案會**自動載入原影片**，不需再手動選取。
- **存檔/匯出走原生對話框**：直接寫入你選的位置（非瀏覽器下載資料夾）。

## 二、網頁版（免安裝）

- 雙擊 **`啟動工具.bat`**：首次自動 ### 桌面版

```bash
npm install          # 第一次安裝
npm run electron     # build + 啟動 Electron
# 或雙擊 啟動桌面版.bat
```

> **提示**：桌面版打包或安裝後，作業系統會將 `.subtool` 專案檔關聯至此程式。往後只要直接雙擊 `.subtool` 專案檔，即可開啟軟體並自動載入該專案進度！。
  （本專案已改為 Vite + ES 模組，原始 `index.html` 是開發入口，**不能直接雙擊**；要邊改邊看請用 `npm run dev`，開 http://localhost:8777 熱更新。）
- 原生播放 MP4 / MOV(H.264) / MP3 / WAV（數 GB 串流不佔記憶體）。
- MXF / 多音軌 / 數 GB 大檔受**瀏覽器記憶體上限**限制，請改用桌面版。

## 三、操作（兩版相同）

### 三大區塊

**播放窗**（左上）｜**字幕列表**（右）｜**時間軸**（下）。拖曳區塊之間的**分隔線**可調整字幕列表寬度、時間軸高度。

### 播放窗

- 匯入影片時**自動偵測 FPS**；FPS 可選 **23.98 / 24 / 25 / 29.97（Drop-frame）/ 29.97（Non-Drop）/ 30**。時間以「**時:分:秒:格**」(例 `01:05:28:16`) 顯示；29.97 Drop-frame 以分號分隔格數（例 `01:05:28;16`）。
- 控制鍵依序：後退 5 秒、後退 1 秒、退 1 格、播放/暫停、進 1 格、前進 1 秒、前進 5 秒。
- **JKL 穿梭輪**：`J` 倒帶、`K` 暫停、`L` 正播；連按 J/L 逐級加速（1× → 1.5× → 2× …每階 0.5×，**最高 5×**）。
- 在影片畫面上**按右鍵** → 切換**音軌**、調整**播放速度**。
- **雙擊時間數字**可直接輸入時間：例 `01235512`→`01:23:55:12`、`35722`→`00:03:57:22`、`8:13`→`00:00:08:13`；超過片長會提示「超過片長」且不跳轉。**右鍵時間數字**可複製時間碼。
- 多軌道在同一時間都有字幕時會**疊加全部顯示**，各軌位置由「字幕列表 → 樣式區」的垂直位置自行調整。

### 上字幕流程

1. 選一條字幕（或用 `Shift+↑ / ↓` 在所選字幕上/下方新增空白字幕）。
2. 播放到開始處按 `I`、結束處按 `O`（設定**被選字幕**的起訖）。
3. 在列表輸入文字，或**雙擊時間軸字幕區塊**直接編輯。

也可先匯入「純文字」聽打稿，再逐句用 `I`／`O` 套時間。

> **空白專案快速開始**：無需先載入影片；啟動時已有「軌道 1」，可直接移動播放點、按 `I`/`O` 新增空白字幕，再按 `\` 縮放至已建字幕範圍。

### 字幕列表

- 上方**下拉選單**：切換顯示的軌道（或全部）。
- 下方**樣式區**：調整該軌道的字幕**大小／垂直位置／對齊**（即時反映在預覽）。
- 點選字幕：`Ctrl`+點=多選、`Shift`+點=範圍選；右鍵可把選取（含多選）**移到其他軌道**。
- 內容超出畫面時，旁邊/下方自動出現捲動條。

### 時間軸（多軌道）

- **時間尺/波形**按住拖曳＝移動播放點；點軌道**空白處**＝取消所有選取。
- **只有按住左鍵不放拖曳**才會框選多句字幕。
- 拖曳字幕區塊＝移動（上下拖可**換軌道**）；拖左右邊緣＝調整起訖。拖移時會**磁吸**到播放點與其他字幕的邊界，且同軌字幕**時間碼不會重疊**。
- 左欄：`＋軌道` 新增、`✕` 刪除、`👁` 顯示/隱藏、**雙擊名稱可改名**；軌道過多時側邊出現上下捲動條。
- 每軌都可放字幕，**匯出時可勾選要輸出哪些軌道**。
- **🕘 動作紀錄**：開啟清單，可逐步復原(`Ctrl+Z`)／重做(`Ctrl+Shift+Z`)，點清單任一步可跳到該狀態。
- **📌 備忘錄**：`＋ 新增` 會記下**目前播放時間點**＋文字；每項可**打勾**表示已確認；點該項的**時間**會把播放窗與時間軸跳到該時間點。

### 快捷鍵

（完整對照另見 `shortcuts.csv`）

**播放 / 穿梭**

| 鍵 | 功能 |
|----|------|
| `空白鍵` | 播放 / 暫停 |
| `J` / `K` / `L` | 倒帶 / 暫停 / 正播；連按 J、L 逐級加速（每階 0.5×，**最高 5×**） |
| `←` / `→` | 前 / 後一格 ｜ `Shift` = ±1 秒 ｜ `Ctrl`+`Shift` = ±5 秒 ｜ 長按 = 連續倒/正播 |
| `Home` / `End`、`Shift`+`↑` / `↓` | 跳到開頭 / 片尾 |
| `1` / `2` | 縮小 / 放大時間軸 |
| `7` / `8` / `9` | 開啟或關閉面板：紀錄(7)、備註(8)、字幕檢查(9) |

**字幕 / 選取**

| 鍵 | 功能 |
|----|------|
| `I` / `O`（或 `Q` / `W`） | 設定被選字幕的開始 / 結束點 = 目前播放點 |
| `↑` / `↓` | 跳轉至上一個 / 下一個字幕邊界 |
| `Ctrl`+`↑` / `↓` | 跳到同軌上一句 / 下一句並選取 |
| `Ctrl`+`Shift`+`↑` / `↓` | 跳到本軌第一句 / 最後一句 |
| `D` | 選取目前播放點所在的字幕 |
| `P` | 將選取字幕整批位移到播放點 |
| `Enter` | 編輯選取字幕文字 |
| `Del` | 刪除選取（可多選） |
| `Esc` | 取消選取 / 關閉上字幕模式 |
| `Z` | 切換不覆蓋 / 可覆蓋狀態 |
| `X` | 切換保留 / 刪除狀態 |
| `Ctrl`+`A` | 全選目前軌道 |
| `Ctrl`+`C` / `Ctrl`+`V` | 複製 / 貼上選取字幕 |
| `Ctrl`+`Z` / `Shift`+`Z` | 復原 / 重做 |

**面板 / 編輯 / 檢視**

| 鍵 | 功能 |
|----|------|
| `F` | 打開/關閉字幕檢查面板 |
| `M`（或 `V`） | 新增備忘錄 |
| `Ctrl`+`S` | 儲存 |
| `Ctrl`+`F` | 搜尋 / 取代 |
| `-` / `+` | 縮小 / 放大時間軸 |
| `]` | 適配字幕範圍 |
| `\` | 適配影片長度（或目前已建字幕範圍） |
| `Ctrl`+滾輪 | 縮放時間軸 |

## 四、字幕格式

- **匯入**（自動偵測 UTF-8 / UTF-16 / 含 BOM）：SRT、ASS/SSA、Adobe Encore（`HH:MM:SS:FF`）、純文字。
- **匯出**（一律 **UTF-16 LE 含 BOM**）：SRT、ASS、Adobe Encore、純文字。
- Adobe Encore 是「影格」時碼，請先在右上設定正確 **FPS**（桌面版會自動偵測）。

## 五、檔案清單

| 檔案 | 用途 |
|------|------|
| `啟動工具.bat` | 網頁版：build 後開啟 `dist/index.html` |

| `electron/main.js`、`electron/preload.js` | 桌面版主程序與橋接 |

| `shortcuts.csv` | 完整快捷鍵對照表 |
| `README.md` | 專案入口說明（快速開始） |
| `docs/ARCHITECTURE.md` | 模組架構與技術參考 |
| `docs/ELECTRON_MAINTENANCE.md` | Electron IPC / ffmpeg / mpv 維護指南 |
| `docs/FPS_時碼一致性.md` | FPS 與時碼一致性的設計原則（維護者必讀） |
| `docs/CHANGELOG.md` | 版本歷史 |

### 打包成免裝版（選用）

若要做成免裝 Node 的可攜版本：`npm run dist`（使用 electron-builder，依 `package.json` 設定輸出 **win zip**，解壓即可執行；產物在 `release/`）。
