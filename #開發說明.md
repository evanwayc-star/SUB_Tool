# SUB Tool — 開發 / 建置說明（架構：Vite + ES 模組）

本專案已從「單一 index.html」重構為 **Vite + ES 模組**，但**打包後仍輸出單一可雙擊的 HTML**（用 `vite-plugin-singlefile`），兼顧可維護性與「免安裝、點開即用」。

## 指令
```bash
npm install      # 第一次（安裝 Vite / Electron 等）
npm run dev      # 開發：本機伺服器 + 熱更新 (HMR)，開 http://localhost:8777
npm run build    # 打包成單一檔 dist/index.html（內嵌 JS/CSS，可直接雙擊）
npm run preview  # 預覽打包結果
npm run electron # 打包後以 Electron 桌面版開啟
npm run dist     # 打包 + electron-builder 產生可攜 .exe
```
- 一般使用者：雙擊 `啟動工具.bat`（自動 build 並開啟 dist 單一檔）或 `啟動桌面版.bat`（build + Electron）。

## 目錄結構（已模組化）
```
index.html          Vite 入口（只剩標記 + <script type=module src=/src/main.js>）
src/
  main.js           入口：import styles.css + app.js
  styles.css        全部 CSS
  dom.js            DOM 元素參照（$, video, tlScroll…）
  util.js           純工具：clamp/pad、UTF-8/UTF-16 編解碼、檔案/base64、escapeHTML
  time.js           時間/時碼換算（SRT/ASS/Encore、秒↔時碼）
  formats.js        字幕格式 解析/序列化（SRT / ASS / Encore / TXT）
  state.js          全域狀態 State + 軌道/FPS 模型 + DESK 偵測
  media.js          媒體引擎（影片 + Web Audio 多音軌 + ffmpeg）+ 波形 Wave
  timeline.js       時間軸：尺/波形/軌道/區塊 渲染與 拖曳/框選/縮放 互動
  subtitles.js      字幕列表：渲染、選取(多選)、新增/刪除、軌道切換/樣式
  keyboard.js       鍵盤快捷鍵 + I/O 上字幕 + 上下鍵邊界步進
  project.js        專案存讀 (.subtool)
  menus.js          右鍵選單（播放窗音軌/速度、字幕移軌）
  history.js        動作紀錄（復原 / 重做）
  notes.js          備忘錄
  app.js            協調層(~520 行)：膠合函式、影片事件、doAction、選單接線、匯入匯出、init
electron/
  main.js           Electron 主程序（載入 dist/index.html；--dev 載入 localhost:8777）
  preload.js        contextBridge 暴露 window.subtool（系統 ffmpeg/對話框）
vite.config.mjs     Vite 設定（singlefile 外掛、port 8777）
eslint.config.mjs   開發期 no-undef 靜態檢查（抓跨模組漏匯入）
dist/index.html     打包輸出（單一檔）
```
模組相依為 app.js 作為協調層（hub）：子模組從 `dom/state/util/time/media/timeline/subtitles` 互相 import，並從 `app.js` import 少量膠合函式（setStatus/openModal/renderAll…）。循環 import 由 ES module live-binding 處理，rollup 打包亦驗證。

## 驗證 / 拆分安全網
- `npx eslint src`：抓「使用了未匯入的名稱」(no-undef)。
- `npm run build`：rollup 會抓「import 了某模組未 export 的名稱」。
- 兩者合用即可在拆模組時抓出所有漏接線。

## 模組化完成度
14 個模組（util/time/formats/dom/state/media/timeline/subtitles/keyboard/project/menus/history/notes + app/main）。`app.js` 已縮為 ~520 行的純協調層。

> 開發期 `import.meta.env.DEV` 為真時會掛上 `window.SUB` 除錯把手；正式 build 會自動移除。
